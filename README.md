# Persona Clock ⏰

Persona Clock은 멀티 페르소나 삶을 시각화하고 관리할 수 있게 도와줍니다.

## ✨ 주요 기능

- 🎨 **페르소나 관리**: 나만의 페르소나를 색상과 함께 등록
- ⏱️ **시간 시각화**: 24시간을 원형 시계로 직관적으로 표시
- 📅 **일정 관리**: 시간대별로 어떤 페르소나로 살지 계획
- 🎯 **목표 설정**: 각 시간대마다 달성하고 싶은 목표 기록
- 🌓 **다크/라이트 모드**: 3가지 테마 (라이트/다크/우주) 지원
- 💾 **자동 저장**: 브라우저에 자동으로 데이터 저장

## 🚀 사용 방법

### 노션에 임베드하기

1. 배포된 URL 복사
2. 노션에서 `/embed` 입력
3. URL 붙여넣기

### 로컬에서 실행하기

1. `index.html` 파일 다운로드
2. 브라우저로 파일 열기
3. 끝!

## 📖 사용 가이드

### 1. 페르소나 설정
- 설정 → 페르소나 탭
- 나만의 페르소나 추가 (예: AI Researcher, Builder, Wellness)
- 각 페르소나마다 고유 색상 지정

### 2. 일정 등록
- 설정 → 일정 탭
- 시작 시간 선택
- 종료 시간 선택 (비워두면 자동)
- 페르소나 선택
- 목표 입력 (선택사항)

### 3. 테마 변경
- 설정 → 테마 탭
- ☀️ 라이트 / 🌙 다크 / ✨ 우주 중 선택

### 4. 일정 리셋 시간
- 매일 특정 시간에 일정표 자동 초기화
- 기본값: 새벽 5시
